This project is a Tableau dashboard where I explored microbial genus abundance across ASD-related samples.
I cleaned and reshaped the dataset to make it easier to analyze, then built visualizations like KPI summaries, a Top 10 genus chart, a distribution plot, a heatmap, and sample-wise comparisons.
The goal was to understand which microbes dominate, how abundance is distributed, and how samples differ from each other.
This project reflects my ability to work with raw data, uncover patterns, and present findings clearly through an organized dashboard.
